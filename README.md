# prelegal

A Python project for pre-legal processing and document handling.

## Prerequisites

- Python 3.9+
- `pip` or a virtual environment manager (e.g. `venv`, `pipenv`)

## Setup

```bash
# Clone the repository
git clone git@github.com:sitati-elsis/prelegal.git
cd prelegal

# Create and activate a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```
